Tentang Saya

Halo! Saya Apta Nayottama, seorang siswa yang menekuni keahlian di bidang mekatronika.Saya mulai dengan pemrograman PLC untuk mengotomatisasi sistem industri, merancang logika kontrol untuk mengelola proses produksi secara efisien. Selanjutnya, saya mengerjakan pemrograman web, di mana saya membuat aplikasi berbasis internet untuk mempermudah pengolahan data dan interaksi pengguna menggunakan HTML, CSS, dan JavaScript. Terakhir, saya mengembangkan proyek IoT, menghubungkan perangkat fisik dengan internet menggunakan platform seperti Arduino dan Raspberry Pi, serta integrasi dengan aplikasi berbasis web untuk kontrol jarak jauh. Proyek-proyek ini memberi saya pengalaman praktis dalam menciptakan solusi otomatisasi yang efisien dan terhubung.
